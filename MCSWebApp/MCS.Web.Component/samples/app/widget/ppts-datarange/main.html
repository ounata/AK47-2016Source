示例代码

Html及Javascript 
                        Html

                        默认用法：<ppts-datarange min="vm.minAmount" max="vm.maxAmount" /><br/>
                        更换单位：<ppts-datarange min="vm.minAmount" max="vm.maxAmount" unit="万元"/><br />
                        修改默认显示文字：<ppts-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" css="col-xs-4 col-sm-4" /><hr />
                        <span>起始金额：{{vm.minAmount}}, 截止金额：{{vm.maxAmount}}</span>
                        Javascript

                        angular.module('app.widget')
                            .controller('PPTSDatarangeController', ['$scope', function($scope) {
                                var vm = this;
                            }]);

参数说明

属性 用途 是否必选 默认值 示例
min 最小值 是 无 <ppts-datarange min="vm.minAmount" max="vm.maxAmount" />
max 最大值 是 无 <ppts-datarange min="vm.minAmount" max="vm.maxAmount" />
unit 单位 否 元 <ppts-datarange min="vm.minAmount" max="vm.maxAmount" unit="万元"/>
minText 最小值文本框的文字提示 否 "起始金额" <ppts-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" />
maxText 最大值文本框的文字提示 否 "截止金额" <ppts-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" />
css 额外的样式设置 否 无 <ppts-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" css="col-xs-4 col-sm-4" />
