应用范例

默认用法：
增加zIndex(防覆盖)：
会议日期：{{vm.meetingTime}}

示例代码

Html及Javascript 
                        Html

                        默认用法：<ppts-datepicker model="vm.meetingTime" /><br/>
                        增加zIndex(防覆盖)： <ppts-datepicker model="vm.meetingTime" z-index="1060" /><br/>
                        <span>开会日期：{{vm.meetingTime}} </span>
                        Javascript

                        angular.module('app.widget')
                            .controller('PPTSDatepickerController', ['$scope', function ($scope) {
                                var vm = this;
                            }]);

参数说明

属性 用途 是否必选 默认值 示例
model 绑定选中的日期 是 无 <ppts-datepicker model="vm.meetingTime" />
zIndex 增加zIndex防覆盖(主页面和弹出页面交互) 是 无 <ppts-datepicker model="vm.meetingTime" z-index="1060"/>
css 自定义样式 否 无 <ppts-datepicker model="vm.meetingTime" css="col-sm-4"/>
