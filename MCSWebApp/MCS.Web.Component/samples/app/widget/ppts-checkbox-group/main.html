示例代码

Html及Javascript 
                        Html

                        异步数据源：<ppts-checkbox-group category="grade_async" model="vm.selectGrades" /><br/>
                        同步数据源：<ppts-checkbox-group category="grade" model="vm.selectGrades" async="false"/><br />
                        指定清空选项：<ppts-checkbox-group category="grade" model="vm.selectGrades" async="false" clear="vm.selectGrades=[]" /><br />

                        Javascript

                        angular.module('app.widget')
                        .controller('PPTSCheckboxGroupController', ['$scope', '$http', function ($scope, $http) {
                            var vm = this;
                            vm.dictionaries = {
                                'c_codE_ABBR_CUSTOMER_GRADE': [
                                    { key: 'G1', value: '一年级' },
                                    { key: 'G2', value: '二年级' },
                                    { key: 'G3', value: '三年级' },
                                    { key: 'G4', value: '四年级' },
                                    { key: 'G5', value: '五年级' },
                                    { key: 'G6', value: '六年级' }
                                ]
                            };
                            mcs.util.merge(vm.dictionaries);
                            $http.get('app/widget/ppts-checkbox-group/sample.json').then(function (result) {
                                mcs.util.merge(result.data.dictionaries);
                                $scope.$broadcast('dictionaryReady');
                            });
                        }]);

参数说明

属性 用途 是否必选 默认值 示例
category 字典数据源的类别Key(如:grade) 是 无 'grade': 'c_codE_ABBR_CUSTOMER_GRADE'
model 绑定选中的数据值（多个,一般为数组） 是 无 <ppts-checkbox-group category="grade_async" model="vm.selectGrades" />
async 是否异步获取数据源，不设置则为true,当异步获取数据源时需触发广播：$scope.$broadcast('dictionaryReady'); 否 true <ppts-checkbox-group category="grade" model="vm.selectGrades" async="false"/>
clear 是否添加清空选项按钮(可选) 否 无 <ppts-checkbox-group category="grade" model="vm.selectGrades" async="false" clear="vm.selectGrades=[]" />
