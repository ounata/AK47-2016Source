示例代码

Html及Javascript 
                        Html

                        默认用法：<ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime"/><br />
                        带提示：<ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" start-text="会议开始日期" end-text="会议结束日期" /><br />
                        增加zIndex(防覆盖)： <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" z-index="1060"/><br />
                        <span>会议开始日期：{{vm.meetingStartTime}}, 会议结束日期：{{vm.meetingEndTime}} </span>
                        Javascript

                        angular.module('app.widget')
                            .controller('PPTSDaterangepickerController', ['$scope', function ($scope) {
                                var vm = this;
                            }]);

参数说明

属性 用途 是否必选 默认值 示例
startDate 绑定选中的开始日期 是 无 <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime"/>
endDate 绑定选中的结束日期 是 否 <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime"/>
startText 开始日期提示 否 “开始时间” <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" start-text="会议开始日期"/>
endText 结束日期提示 否 “结束时间” <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" end-text="会议结束日期"/>
zIndex 增加zIndex防覆盖(主页面和弹出页面交互) 是 无 <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" z-index="1060"/>
css 自定义控件的样式 是 无 <ppts-daterangepicker start-date="vm.meetingStartTime" end-date="vm.meetingEndTime" css="col-sm-4"/>
