应用范例

接口说明：

  1. 树的容器标签必须使用可唯一标识的id属性，例如：id="userTree"
  2. 树控件初始化以后，在tree setting 对象上添加接口方法，例如：getNodesChecked 获取已经check的所有节点；getNamesOfNodesChecked 获取已经check的节点的名称数组；getIdsOfNodesChecked 获取已经check的节点的id数组
  3. 如果需要改变节点选取框为radio，则请修改树的配置对象 check: { enable: true, chkStyle: 'radio', radioType: "all" }

参数说明

settings:

参数 用途 取值范围 示例
data 定义node数据的约定结构 JSON对象 参见示例
view node显示相关配置 boolean fontCss部分为样式表格式定义字体样式
async 如果采用异步加载子树，配置异步行为 详细参见下方链接页面 autoParam用来配置node的标识字段
callback 定义node事件回调 function 参见示例

TreeNode:

参数 用途 取值范围 示例
open 当前node是否展开 boolean open:true
parent 当前node是否有子节点 boolean parent:false
checked 当前节点是否选中 boolean checked:true
chkDisabled 当前节点是否可选 boolean chkDisabled:true
