应用范例

{{vm.model}}

示例代码

HTML Code
<mcs-cascading-select data="vm.data" ng-model="vm.selection" required="true" />
JS Code
'use strict';
(
function() {
    angular.module('app.component')

    .controller('MCSCascadingSelectController', ['$scope', '$http', function($scope, $http) {
        var vm = {};
        $scope.vm = vm;
        vm.data = {};

        $http.get('../src/tpl/sample.json').success(function(result) {
            vm.data = result;
        });

        vm.selection = {
            province: '',
            city: '',
            district: ''
        }
    }])
}
)();

参数说明

参数 用途 取值 示例
data 设置数据源 json对象 sample
ng-model 绑定到上下文scope的数据对象 js对象 ng-model="vm.address"
required 设置下拉选项是否为必选 boolean值 required="true"
