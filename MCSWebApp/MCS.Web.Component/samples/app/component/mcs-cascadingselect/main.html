参数说明

属性 用途 范围 默认值 示例
level 级联下拉框的层级(最多支持4级) {2,3,4} 3 <mcs-cascading-select level="4" ></mcs-cascading-select>
caption 级联下拉框的文字提示（以“,”分隔，不设置则为“请选择”，如设置将自动计算level） 无 “请选择” <mcs-cascading-select caption="省份,城市,区县" ></mcs-cascading-select>
data 同步数据源，此时需要设置async="false" 无 无 <mcs-cascading-select caption="省份,城市,区县" data="vm.items" async="false" ></mcs-cascading-select>
async 是否异步获取数据源，不设置则为true，此时需要指定url 无 true <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" ></mcs-cascading-select>
url 异步请求的数据源地址 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" ></mcs-cascading-select>
path 指定数据源是否采用链式路径（如：0|0,1|0,1,2），不设置则为false 无 false <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" path="true" ></mcs-cascading-select>
root 数据源的根级节点值，默认为0 无 0 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" path="true" root="-1" ></mcs-cascading-select>
params 数据源请求的参数（以对象形式） 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" ></mcs-cascading-select>
otherParams 数据源请求的附加参数（以对象形式） 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" other-params="vm.otherParams" ></mcs-cascading-select>
parentKey 指定数据源父级别字段名称 无 “parentKey” <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" other-params="vm.otherParams" parent-key="parent"></mcs-cascading-select>
disabledLevel 禁用级联框的层级(可为1,2,3,4) 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" other-params="vm.otherParams" parent-key="parent" disabled-level="2"></mcs-cascading-select>
customStyle 级联控件的自定义样式 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" other-params="vm.otherParams" parent-key="parent" custom-style="width:50%"></mcs-cascading-select>
callback 选中某一级数据触发回调 无 无 <mcs-cascading-select caption="省份,城市,区县" url="/api/customers" params="vm.params" other-params="vm.otherParams" parent-key="parent" custom-style="width:50%" callback="vm.select()"></mcs-cascading-select>
