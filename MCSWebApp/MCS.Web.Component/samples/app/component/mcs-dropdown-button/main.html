参数说明

属性 用途 范围 默认值 示例
title 设置下拉按钮的标题文字显示（如：订购） 无 <mcs-dropdown-button title="订 购" items="vm.products"></mcs-dropdown-button>
items 设置下拉按钮的下拉内容来源（数组） 无 <mcs-dropdown-button title="订 购" items="vm.products"></mcs-dropdown-button>
icon 下拉按钮无法满足时,通用下拉按钮的图标如(“订购”图标可使用“shopping-cart”或“fa-shopping-cart”) 无 fa-plus <mcs-dropdown-button title="订 购" items="vm.products" icon="shopping-cart"></mcs-dropdown-button>
category 设置下拉按钮的类型 "info", "primary", "warning", "success", "danger" info <mcs-dropdown-button title="订 购" items="vm.products"></mcs-dropdown-button>
