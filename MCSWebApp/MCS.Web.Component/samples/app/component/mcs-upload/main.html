应用范例

带附件的表单提交

示例代码

 <div files-upload error-message="只能选择图片或word文档" model="vm.userInfo.files" pattern="{size: {max: '5MB'},pattern: 'image/*,.docx'}" readonly="false" resource-Id="{{vm.userInfo.resourceId}}" files-amount="5" module-name="{{vm.userInfo.moduleName}}" download-url="http://localhost/MCSWebApp/MCS.Web.API/api/sample/DownloadMaterial" url="http://localhost/MCSWebApp/MCS.Web.API/api/sample/UploadMaterialWriteDB">
 </div>

参数说明

参数 用途 取值范围 示例
error-message 设定文件类型选择错误的提示消息 string error-message="只能选择图片"
files-amount 设定上传的附件数量 number files-amount="5"
module-name 当前操作所属模块名称 string module-name="customer"
download-url 附件下载接口 string download-url="http://localhost/api/downloadFile"
url 附件上传地址 string url="http://localhost/api/fileupload"
resource-Id 能唯一标识当前业务的ID string resource-Id="1122-2233-3322"
pattern 文件类型验证模式 string pattern="image/*,.docx"
