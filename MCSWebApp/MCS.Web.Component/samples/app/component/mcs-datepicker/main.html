示例代码

Html及Javascript 
                        Html

                        默认用法：<mcs-datepicker model="vm.meetingTime" /><br/>
                        带提示： <mcs-datepicker model="vm.meetingTime" placeholder="会议日期" /><br/>
                        禁用： <mcs-datepicker model="vm.meetingTime" placeholder="会议日期" disabled="true" /><br/>
                        增加zIndex(防覆盖)： <mcs-datepicker model="vm.meetingTime" z-index="1060" /><br/>
                        <span>开会日期：{{vm.meetingTime}} </span>
                        Javascript

                        angular.module('app.component')
                            .controller('MCSDatepickerController', ['$scope', function ($scope) {
                                var vm = this;
                            }]);

日期类型数据手动格式化

项目中如果有日期类型数据手动格式化的需求，可以使用本项目中引入的moment.js库，使用示例如下： moment(d).format('YYYY-MM-DD'); 参考API: http://momentjs.cn/docs/#/displaying

参数说明

属性 用途 是否必选 默认值 示例
model 绑定选中的日期 是 无 <mcs-datepicker model="vm.meetingTime" />
placeholder 日期控件提示文本 否 "输入日期" <mcs-datepicker model="vm.meetingTime" placeholder="会议日期"/>
startDate 绑定开始日期 是 无 <mcs-datepicker start-date="vm.meetingStartDate"/>
endDate 绑定截止日期 是 否 <mcs-datepicker end-date="vm.meetingEndDate"/>
disabled 禁用当前日期控件 否 false <mcs-datepicker model="vm.meetingTime" placeholder="会议日期" disabled="true"/>
zIndex 增加zIndex防覆盖(主页面和弹出页面交互) 是 无 <mcs-datepicker model="vm.meetingTime" z-index="1060"/>
css 自定义样式 否 无 <mcs-datepicker model="vm.meetingTime" css="col-sm-4"/>
