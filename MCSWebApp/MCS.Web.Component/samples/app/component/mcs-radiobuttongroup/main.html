应用范例

Student:

已选中: {{vm.parent.gender}}

Parent:

已选中: {{vm.customer.gender}}

示例代码

Html及Javascript 
                        Html

                        <mcs-radiobutton-group data="vm.dictionaries.genders" model="vm.parent.gender" />
                        Javascript

                        angular.module('app.component').controller('MCSRadiobuttonGroupController', ['$scope', function ($scope) {
                            var vm = this;
                            vm.parent = {
                                gender: 'G1'
                            };
                            vm.customer = {
                                gender: 'G2'
                            };
                            vm.dictionaries = {
                                genders: [{
                                    key: 'G1',
                                    value: '男'
                                }, {
                                    key: 'G2',
                                    value: '女'
                                }, {
                                    key: 'G3',
                                    value: '未知'
                                }]
                            };
                             // 一般情况下选完之后不会做其他事，适用于联动响应的情况
                            $scope.$watch('vm.parent.gender', function () {
                                alert('我选中了' + vm.parent.gender + '，但是我还想干点别的，来这里吧。');
                            });
                        }

参数说明

属性 用途 范围 默认值 示例
data 设置复选框数据源, 必须为数组结构且包含{key,value}的形式 无 无 <mcs-radiobutton-group data="vm.dictionaries.genders" />
model 设置需要绑定的字段 无 无 <mcs-radiobutton-group data="vm.dictionaries.genders" model="vm.parent.gender"/>
