单选列表框

已选中: {{vm.checkresult.selectedGender}}

代码 
                        Html

                        <mcs-radiobutton-group category="selectedGender" data="vm.dictionaries.genders" click="vm.select(category, key, eventargs)" defaultValue="G1"/>
                        Javascript

                        angular.module('app.component').controller('MCSRadiobuttonGroupController', [
                            '$scope', function ($scope) {
                                var vm = this;
                                vm.checkresult = {
                                    selectedGender: []
                                };
                                vm.dictionaries = {
                                    genders: [
                                        { key: 'G1', value: '男', checked: false },
                                        { key: 'G2', value: '女', checked: true }, //默认选中
                                        { key: 'G3', value: '未知', checked: false }
                                    ]
                                };
                                vm.select = function (category, key, eventargs) {
                                    mcs.setSelectedItems(vm.checkresult[category], key, eventargs);
                                };
                            }
                        ]);

使用说明

1. 指定复选框所属类别category(如“selectedGender”, 需与controller中接受选中结果的名字对应);

2. 指定复选框数据源data, 必须为数组结构且包含{key,value}的形式;

3. 如需要接受选中项结果, 必须设置click属性且指定vm.select(category, value, eventargs),事件在controller中定义;

4. 如需要默认选中某项目, 可以设置defaultValue为选中项的key值或改变选中项的checked值为true。
