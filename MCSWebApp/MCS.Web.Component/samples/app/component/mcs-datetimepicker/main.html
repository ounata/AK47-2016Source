示例代码

Html及Javascript 
                        Html

                        默认用法：<mcs-datetimepicker model="vm.meetingTime" /><br/>
                        带提示： <mcs-datetimepicker model="vm.meetingTime" placeholder="会议时间" /><br/>
                        禁用： <mcs-datetimepicker model="vm.meetingTime" placeholder="会议时间" disabled="true"/><br/>
                        增加zIndex(防覆盖)： <mcs-datetimepicker model="vm.meetingTime" z-index="1060"/><br/>
                        <span>开会日期：{{vm.meetingTime}} </span>
                        Javascript

                        angular.module('app.component')
                            .controller('MCSDatetimepickerController', ['$scope', function ($scope) {
                                var vm = this;
                            }]);

参数说明

属性 用途 是否必选 默认值 示例
model 绑定选中的日期 是 无 <mcs-datetimepicker model="vm.meetingTime" />
placeholder 时间控件提示文本 否 "输入时间" <mcs-datetimepicker model="vm.meetingTime" placeholder="会议时间"/>
startDate 绑定开始时间 是 无 <mcs-datetimepicker start-date="vm.meetingStartTime"/>
endDate 绑定截止时间 是 否 <mcs-datetimepicker end-date="vm.meetingEndTime"/>
disabled 禁用当前时间控件 否 false <mcs-datetimepicker model="vm.meetingTime" placeholder="会议时间" disabled="true"/>
zIndex 增加zIndex防覆盖(主页面和弹出页面交互) 是 无 <mcs-datetimepicker model="vm.meetingTime" z-index="1060"/>
css 自定义样式 否 无 <mcs-datetimepicker model="vm.meetingTime" css="col-sm-4"/>
