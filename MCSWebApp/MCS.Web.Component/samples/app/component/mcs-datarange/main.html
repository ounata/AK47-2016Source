示例代码

Html及Javascript 
                        Html

                        默认用法：<mcs-datarange min="vm.minAmount" max="vm.maxAmount" /><br/>
                        设置数据类型：<mcs-datarange min-text="最小年龄" max-text="最大年龄" min="vm.minAge" max="vm.maxAge" datatype="int" /><br/>
                        无单位：<mcs-datarange min="vm.minAmount" max="vm.maxAmount" unit="" /><br/>                     
                        更换单位：<mcs-datarange min="vm.minAmount" max="vm.maxAmount" unit="万元"/><br />
                        禁用模式：<mcs-datarange min="vm.minAmount" max="vm.maxAmount" disabled="true"/><br />
                        修改默认显示文字：<mcs-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" css="col-xs-4 col-sm-4" /><hr />
                        <span>起始金额：{{vm.minAmount}}, 截止金额：{{vm.maxAmount}}</span>
                        Javascript

                        angular.module('app.component')
                            .controller('MCSDatarangeController', ['$scope', function($scope) {
                                var vm = this;
                            }]);

参数说明

属性 用途 是否必选 默认值 示例
min 最小值 是 无 <mcs-datarange min="vm.minAmount" max="vm.maxAmount" />
max 最大值 是 无 <mcs-datarange min="vm.minAmount" max="vm.maxAmount" />
unit 单位 否 元 <mcs-datarange min="vm.minAmount" max="vm.maxAmount" unit="万元"/>
minText 最小值文本框的文字提示 否 "起始金额" <mcs-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" />
maxText 最大值文本框的文字提示 否 "截止金额" <mcs-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" />
datatype 设置文本框的数据类型 int,number number <mcs-datarange min-text="最小年龄" max-text="最大年龄" min="vm.minAge" max="vm.maxAge" datatype="int" />
disabled 设置禁用模式 true,false false <mcs-datarange min="vm.minAmount" max="vm.maxAmount" disabled="true"/><br />
css 额外的样式设置 否 无 <mcs-datarange min="vm.minAmount" max="vm.maxAmount" min-text="充值金额起" max-text="充值金额止" css="col-xs-4 col-sm-4" />
