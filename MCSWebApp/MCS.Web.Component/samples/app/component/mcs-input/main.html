参数说明

属性 用途 范围 默认值 示例
model 绑定文本框跟后台的数据值如(model="vm.parentName") 无 无 <mcs-input model="vm.parentName" />
placeholder 设置文本框的默认显示文本 无 无 <mcs-input id="parentName" placeholder="家长姓名" model="vm.parentName" />
type 设置文本框的类型 text, phone, email.... text <mcs-input id="parentName" placeholder="家长联系方式" type="phone" model="vm.parentName" />
css 设置文本框的样式 无 无 input.hightlight { background-color: #11a9cc; }
<mcs-input placeholder="家长姓名" css="hightlight" model="vm.parentName" />
datatype 设置文本框的数据类型 int,number,string string <mcs-input placeholder="充值金额" model="vm.amount" datatype="number" /> <mcs-input placeholder="学员年龄" model="vm.age" datatype="int" />
