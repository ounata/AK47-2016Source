打印说明：

本项目打印功能兼容包括IE/Edge/Chrome/Firefox，使用步骤：
  • 在页面的Controller中引入'printService'服务。
  • 使用Div="printArea" 来标识出要打印的区域。
  • 在页面适当位置添加打印按钮，按钮点击事件ng-click="vm.print()"，完成打印。

打印实例

这里是要打印的内容啦！

示例代码

HTML

    <div id="printArea">
        <p>这里是要打印的内容啦！</p>
    </div>

JS

    angular.module('app.component').controller('MCSDatatableController', [
        '$scope', 'printService',
        function($scope, $q, $timeout, printService) {
            var vm = this;            

            vm.print = function() {
                printService.print();
            }
        }
    ]);
