参数说明

属性 用途 范围 默认值 示例
type 原有按钮的属性(如：submit, button) "submit", "button".... button <mcs-button type="submit"></mcs-button>
category 设置通用按钮的类型(带有默认文本和图标, 但定制按钮可以不设置type)如(“新 增”,“编 辑”,“删 除”,“查 询”,“保 存”, “取 消”等) "add", "edit", "delete", "search", "save", "cancel".... 无 <mcs-button category="add"></mcs-button>
text 通用按钮无法满足时,设置通用按钮的文本如(“新增潜在客户”) 无 无 <mcs-button category="add" text="新增潜在客户"></mcs-button>
icon 通用按钮无法满足时,设置通用按钮的图标如(“人民币”图标可使用“cny”或“fa-cny”) 无 无 <mcs-button category="add" text="充 值" icon="cny"></mcs-button>
size 通用按钮无法满足时,设置通用按钮的大小如(默认按钮不设置大小) "mini", "medium", "large" 无 <mcs-button category="add" text="我比你们大" icon="cny" size="large"></mcs-button>
css 通用按钮无法满足时,设置通用按钮的样式如(默认按钮不设置大小) 无 无 .max-button { width: 200px; height: 100px; background-color: #660e0b; }
<mcs-button category="add" text="没有最大,只有更大" icon="cny" size="large" css="max-button"></mcs-button>
