参数说明

属性 用途 范围 默认值 示例
text 设置通用标签的文本如(“学生家长”等) 无 无 <mcs-label text="学生家长"/>
required 设置是否为必填项 无 无 <mcs-label required="true" text="学生家长"/>
for-input 关联到一个文本框,值为文本框id(点击标签可以将焦点移到文本框) 无 无 <mcs-label required="true" text="学生家长" for-input="parentName"/> <mcs-input id="parentName"/>
css 设置标签的样式 无 无 label.hightlight { background-color: #11a9cc; }
<mcs-label required="true" text="学生家长" for-input="parentName" css="hightlight"/> <mcs-input id="parentName"/>
