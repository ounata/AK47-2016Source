Html及Javascript 
                        Html

                        异步数据源：<mcs-select category="grade_async" model="vm.selectGrade" /><br />
                        同步数据源：<mcs-select category="grade" model="vm.selectGrade" async="false" /><br />
                        指定标题：<mcs-select category="grade" model="vm.selectGrade" async="false" caption="选择年级"/><br />
                        执行回调：<mcs-select category="grade" model="vm.selectGrade" async="false" callback="vm.select(item, model)" /><br />

                        Javascript

                        angular.module('app.component')
                        .controller('MCSSelectController', ['$scope', '$http', function ($scope, $http) {
                            var vm = this;
                            vm.dictionaries = {
                                'c_codE_ABBR_CUSTOMER_GRADE': [
                                    { key: 'G1', value: '一年级' },
                                    { key: 'G2', value: '二年级' },
                                    { key: 'G3', value: '三年级' },
                                    { key: 'G4', value: '四年级' },
                                    { key: 'G5', value: '五年级' },
                                    { key: 'G6', value: '六年级' }
                                ]
                            };
                            mcs.util.merge(vm.dictionaries);
                            $http.get('app/widget/sample.json').then(function (result) {
                                mcs.util.merge(result.data.dictionaries);
                                $scope.$broadcast('dictionaryReady');
                            });
                            vm.select = function (item, model) {
                                alert('我选中了item：' + item.value + ',对应key:' + model);
                            };
                        }]);

参数说明

属性 用途 范围 默认值 示例
category 字典数据源的类别Key(如:grade) 是 无 'grade': 'c_codE_ABBR_CUSTOMER_GRADE'
model 绑定选中的数据值（单值） 是 无 <mcs-select category="grade_async" model="vm.selectGrade" />
async 是否异步获取数据源，不设置则为true,当异步获取数据源时需触发广播：$scope.$broadcast('dictionaryReady'); 否 true <mcs-select category="grade" model="vm.selectGrade" async="false"/>
caption 显示默认的提示标题 否 "请选择" <mcs-select category="grade" model="vm.selectGrade" async="false" caption="选择年级"/>
callback 选中之后执行回调 否 无 <mcs-select category="grade" model="vm.selectGrade" async="false" callback="vm.select(item, model)" />
css/style 添加自定义样式 否 无 <mcs-select category="grade" model="vm.selectGrade" async="false" css="col-sm-4 col-xs-4 pull-right" style="width:400px;"/>
