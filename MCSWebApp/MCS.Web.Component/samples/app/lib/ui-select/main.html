示例代码

Html及Javascript 
                        Html

                        <mcs-button click="vm.enable()" text="启用"></mcs-button>
                        <mcs-button click="vm.disable()" text="禁用"></mcs-button>
                        <mcs-button click="vm.clear()" text="清空"></mcs-button>
                        <p>已选中: {{vm.customer.entranceGrade}}</p>
                        <ui-select ng-model="vm.customer.entranceGrade" ng-disabled="vm.disabled" theme="selectize" style="width:400px;">
	                        <ui-select-match placeholder="请选择">{{$select.selected.value}}</ui-select-match>
	                        <ui-select-choices repeat="item.key as item in vm.dictionaries.grades | props: {value: $select.search}">
		                        <span ng-bind-html="item.value | highlight: $select.search"></span>
	                        </ui-select-choices>
                        </ui-select>
                        Javascript

                        angular.module('app.lib').controller('AppUISelectController', [
                            '$scope', function ($scope) {
                                var vm = this;
                                vm.disabled = undefined;
                                vm.enable = function () {
                                    vm.disabled = false;
                                };
                                vm.disable = function () {
                                    vm.disabled = true;
                                };
                                vm.clear = function () {
                                    vm.customer.entranceGrade = '';
                                };
                                vm.customer = {
                                    entranceGrade: ''
                                };
                                vm.dictionaries = {
                                    grades: [
                                        { key: 'G1', value: '一年级' },
                                        { key: 'G2', value: '二年级' },
                                        { key: 'G3', value: '三年级' },
                                        { key: 'G4', value: '四年级' },
                                        { key: 'G5', value: '五年级' },
                                        { key: 'G6', value: '六年级' }
                                    ]
                                };
                            }
                        ]);

参数说明

属性 用途 范围 默认值 示例
data 设置复选框数据源, 必须为数组结构且包含{key,value}的形式 无 无 <mcs-radiobutton-group data="vm.dictionaries.genders" />
model 设置需要绑定的字段 无 无 <mcs-radiobutton-group data="vm.dictionaries.genders" model="vm.parent.gender"/>
